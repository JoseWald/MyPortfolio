ESTI WEB (November 2024 - January 2025)

Web Suggestion for ESTI's Web Page Project As part of our integration project at ESTI, my team and I developed a dynamic web page. During the development phase, I worked as the Laravel developer, which allowed me to implement several dynamic features. One key functionality is that the admin can now log into the website to post college events. Additionally, the admin can delete or update events in case of mistakes. This feature ensures that the content remains accurate and up-to-date. Furthermore, the admin has the ability to update their login email and password, adding an extra layer of security and flexibility in case of credential management issues.
