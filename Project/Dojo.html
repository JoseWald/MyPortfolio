DOJO (October 2024)

My first Larvel project that I made after following an 8h crash course video.
I call it Dojo beacause of this app , every Ninja PHP (PHP lover) can see a job annoucement that need PHP coding skill. That is a project similar to Linkedin but every job annoucement should require a knowledge in PHP. Key Features:
✅ Job listings exclusively for PHP developers
✅ Built with Laravel + MySQL
✅ Simple and intuitive job posting system
✅ A mini-LinkedIn but PHP-focused

  • Github: https://github.com/JoseWald/DOJO
